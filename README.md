# System Log Analyzer & Report Generator

This is a CLI-based Python tool designed to parse system log files, count log levels (INFO, WARNING, ERROR), and generate a detailed summary report.

## Features
- Load and parse `.log` or `.txt` files using Regex.
- Filter logs by level or search by specific keywords.
- Generate and save reports automatically.

## How to Run
1. Open the folder in VS Code.
2. Run the following command in the terminal:
   ```bash
   python log_analyzer.py
